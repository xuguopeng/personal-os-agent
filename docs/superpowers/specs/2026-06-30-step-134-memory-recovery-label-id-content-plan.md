# Step 134：恢复按钮同时显示短 ID 和内容

## 目标

当记忆审核错误日志提供短 ID 恢复命令时，按钮标签同时显示短 ID 和候选内容片段，例如 `批准：#abcd1234 / 我喜欢...`。这样用户既能看到稳定 ID，也能确认具体候选内容。

## 小步骤

1. 从模糊错误输出中解析 `#短ID -> 候选内容`。
2. 生成恢复按钮标签：
   - 短 ID 命令：`批准：#短ID / 内容片段`
   - 内容片段命令：保留现有 `批准：内容片段`
3. 内容片段过长时截断，避免布局被撑开。
4. 构建、Rust 检查、浏览器验证。

## 验证

- `CI=true pnpm build`
- `cargo check`
- `cargo test`
- 浏览器验证：
  - 触发记忆候选模糊命中错误。
  - 错误日志恢复按钮显示 `#短ID / 内容片段`。
  - 点击“填入”仍能把短 ID 命令放入聊天输入框。
  - 页面控制台没有应用错误。
