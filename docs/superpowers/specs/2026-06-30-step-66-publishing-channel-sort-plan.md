# Step 66: 发布渠道排序

## 目标

让发布渠道列表支持排序。渠道变多后，可以按最近发布、发布记录数量、名称、类型和启用状态快速找到目标渠道。

## 小步骤

1. 新增 `PublishingChannelSortMode` 类型。
2. `PublishingChannelsPanel` 增加排序选择器。
3. 排序选项：
   - 最近发布
   - 发布记录多到少
   - 渠道名称
   - 渠道类型
   - 启用状态
4. 列表使用筛选后的排序结果。
5. 保留现有搜索、类型筛选、启用状态筛选。
6. 验证：
   - `CI=true pnpm build`
   - `cargo check`
   - `cargo test`
   - 浏览器检查：设置页可见渠道排序控件，无 console error。

## 非目标

- 不新增后端接口。
- 不改变渠道数据结构。
- 不改变发布记录数据。
