# Step 116：最近任务只看待确认任务

## 目标

在流程日志左侧“最近任务”里增加一个轻量入口，方便只看当前已加载任务中包含真实待确认动作的任务。第一版不新增后端索引，只基于当前任务步骤和任务标题做可见筛选。

## 小计划

1. 在 ProcessLog 左侧任务筛选区域增加“只看待确认”开关。
2. 识别待确认任务：
   - 当前选中任务如果有 actionable pending confirmation，则保留。
   - 任务标题或当前步骤中包含待确认删除线索时保留。
3. 开关只影响左侧最近任务列表，不改变右侧步骤显示。
4. 没有匹配任务时显示空状态。
5. 后续再升级为后端按步骤聚合查询。

## 验证

- `CI=true pnpm build`
- `cargo check`
- `cargo test`
- 浏览器验证：
  - 生成一个删除待确认任务。
  - 打开“只看待确认”后，最近任务列表仍能看到该任务。
  - 取消开关后恢复普通最近任务列表。
  - console error 为空。
