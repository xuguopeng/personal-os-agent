# Step 131：按短 ID 审核记忆候选

## 目标

让记忆候选支持更稳定的审核命令：`@记忆 批准候选 #短ID` / `@记忆 拒绝候选 #短ID`。内容片段可能重复，短 ID 可以作为精确定位方式。

## 小步骤

1. 在 `@记忆 待确认` 回复中显示候选短 ID。
2. 候选列表推荐短 ID 审核命令：
   - `@记忆 批准候选 #xxxx`
   - `@记忆 拒绝候选 #xxxx`
3. 扩展审核意图解析，提取 `#短ID`。
4. 审核候选时优先按短 ID 匹配，其次按内容片段，最后按序号。
5. 构建、Rust 检查、浏览器验证。

## 验证

- `CI=true pnpm build`
- `cargo check`
- `cargo test`
- 浏览器验证：
  - 创建一条记忆候选。
  - 输入 `@记忆 待确认` 后看到短 ID 命令。
  - 输入短 ID 批准命令后成功写入长期记忆。
  - 页面控制台没有应用错误。
