# Step 64: 记录发布优先使用启用渠道

## 目标

让“记录发布”表单优先选择已启用的发布渠道，并把停用渠道清楚标识出来。停用渠道仍然可以用于补录历史，但不会被默认推荐。

## 小步骤

1. `startPublishingRecord` 优先选择同类型且已启用的渠道。
2. 没有启用渠道时，再回退到同类型停用渠道。
3. 发布渠道下拉按启用状态分组：
   - 已启用渠道
   - 已停用渠道
4. 停用渠道选项标注“已停用”。
5. 无渠道时继续使用草稿渠道类型兜底。
6. 验证：
   - `CI=true pnpm build`
   - `cargo check`
   - `cargo test`
   - 浏览器检查：记录发布表单可见渠道选择，无 console error。

## 非目标

- 不阻止选择停用渠道。
- 不删除渠道。
- 不修改发布记录结构。
- 不真实发布外部平台。
