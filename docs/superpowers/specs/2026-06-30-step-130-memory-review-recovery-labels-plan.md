# Step 130：记忆审核恢复命令显示候选片段

## 目标

当记忆审核错误日志里出现多条恢复命令时，按钮标签不再都叫“批准候选”或“拒绝候选”，而是显示候选片段，例如 `批准：我喜欢...`。这样用户能清楚知道每个按钮对应哪条候选。

## 小步骤

1. 从恢复命令中提取 `《...》` 内容片段。
2. 生成更具体的按钮标签：
   - `批准：片段`
   - `拒绝：片段`
3. 片段过长时截断，避免按钮布局被撑坏。
4. 构建、Rust 检查、浏览器验证。

## 验证

- `CI=true pnpm build`
- `cargo check`
- `cargo test`
- 浏览器验证：
  - 触发记忆候选内容片段多命中错误。
  - 错误日志恢复按钮显示候选片段。
  - 点击“填入”仍能把对应命令放入聊天输入框。
  - 页面控制台没有应用错误。
