# Step 22: 任务标题编辑

## 目标

让底部流程日志里的任务标题可以手动改名。自动生成的聊天任务标题通常较长，用户需要能把它整理成清晰的任务名称，方便之后搜索、筛选和回看。

## 小步骤

1. 复用后端 `update_task_session`：
   - 已支持 `title` 和 `status` 更新。
   - 补充测试断言标题更新和状态校验。
2. 前端底部任务详情：
   - 标题旁增加编辑按钮。
   - 编辑状态显示输入框、保存、取消。
   - 空标题不保存。
3. 同步策略：
   - 保存后刷新任务列表。
   - 保持当前任务选中。
   - 任务筛选条件不清空。
4. 验证：
   - `CI=true pnpm build`
   - `cargo check`
   - `cargo test`
   - 浏览器检查：编辑标题后，任务详情和左侧任务列表同步更新。

## 非目标

- 不删除任务。
- 不批量重命名。
- 不调用真实发布、导出、生成或外部服务。
