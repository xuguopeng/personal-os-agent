# Step 65: 发布渠道筛选

## 目标

让发布渠道列表支持搜索和筛选。渠道变多后，用户可以按名称、账号、endpoint、渠道类型和启用状态快速找到目标渠道。

## 小步骤

1. `PublishingChannelsPanel` 增加本地筛选状态。
2. 支持搜索：
   - 渠道名称
   - 账号/站点标识
   - endpoint
   - 默认分类
   - 默认标签
3. 支持渠道类型筛选。
4. 支持启用状态筛选。
5. 列表展示筛选后的结果。
6. 无匹配时展示空状态。
7. 验证：
   - `CI=true pnpm build`
   - `cargo check`
   - `cargo test`
   - 浏览器检查：设置页可见渠道筛选控件，无 console error。

## 非目标

- 不新增后端接口。
- 不改变渠道数据结构。
- 不删除渠道或密钥。
- 不真实发布外部平台。
