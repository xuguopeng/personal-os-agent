# Step 98：聊天搜索博客草稿

## 目标

让 `@博客` 聊天指令可以搜索或列出本地博客草稿，例如 `搜索包含 Agent 的草稿`、`列出待发布草稿`、`有哪些公众号草稿`。

## 小步骤

1. 在 Agent 任务结果里增加草稿搜索结果字段。
2. 增加搜索意图识别，支持搜索、查找、列出、有哪些草稿、草稿列表。
3. 增加搜索条件解析：关键词、状态、渠道类型。
4. 从最新草稿列表里筛选并排序，最多返回 5 条结果。
5. 追加 `publishing_draft_search` 流程步骤，记录查询条件和命中结果。
6. 在聊天回复里输出搜索摘要。
7. 跑前端构建、Rust check/test，并用浏览器验证搜索草稿。

## 范围

- 只读取本地草稿，不修改任何数据。
- 不自动打开外部平台。
- 搜索命中标题、正文、状态、渠道、分类和标签。

## 验收

- 输入 `@博客 搜索《关键词》草稿` 后，聊天回复显示命中草稿摘要。
- 底部流程日志出现 `publishing_draft_search` 和 `agent.blog.draft.search`。
- 构建、Rust 检查和测试通过。
