# Step 125：记忆候选列表显示内容匹配命令

## 目标

让 `@记忆 待确认` 的回复优先给出更安全的内容匹配审核命令。候选列表多时，用户可以直接复制 `@记忆 批准候选《内容片段》`，减少只靠序号带来的误操作风险。

## 小步骤

1. 更新候选列表回复格式。
2. 每条候选显示内容匹配命令：
   - `@记忆 批准候选《...》`
   - `@记忆 拒绝候选《...》`
3. 内容片段过长时截断，保证聊天回复可读。
4. 保留序号展示，但不再把序号命令作为主推荐。
5. 构建、Rust 检查、浏览器验证。

## 验证

- `CI=true pnpm build`
- `cargo check`
- `cargo test`
- 浏览器验证：
  - 创建一条唯一记忆候选。
  - 输入 `@记忆 待确认`。
  - 回复中出现 `@记忆 批准候选《唯一内容》` 和 `@记忆 拒绝候选《唯一内容》`。
  - 页面控制台没有应用错误。
