# Step 106：待确认项复制确认命令

## 目标

让“待确认操作”不只是可见，还能给出下一步该怎么说。第一版只提供复制聊天命令，不提供一键执行，避免误触删除、发布、付费生成或导出。

## 小计划

1. 为待确认步骤生成建议命令：
   - 删除博客草稿：`@博客 确认删除《标题》草稿`
   - 其他确认门：显示通用确认提示，不生成危险执行按钮。
2. 在待确认卡片里展示“建议输入”。
3. 增加复制按钮，把建议命令复制到剪贴板。
4. 复制成功后短暂显示“已复制”，失败时显示“复制失败”。
5. 保持确认流程仍由聊天触发，不在列表里直接执行。

## 验证

- `CI=true pnpm build`
- `cargo check`
- `cargo test`
- 浏览器验证：
  - 普通删除草稿后，待确认卡片显示建议命令。
  - 点击复制按钮后显示已复制。
  - console error 为空。
