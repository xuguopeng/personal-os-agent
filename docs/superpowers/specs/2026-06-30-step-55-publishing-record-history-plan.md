# Step 55: 发布记录历史列表

## 目标

让已经保存的手动发布记录可以集中查看和筛选，不只显示在草稿卡片里的最近一次记录。用户之后查“哪篇博客发到哪个网站或公众号”时，可以直接在博客模块看到历史。

## 小步骤

1. 新增 `PublishingRecordsPanel` 组件。
2. 展示字段：
   - 草稿标题
   - 发布渠道
   - 发布 URL
   - 状态
   - 备注
   - 发布时间
3. 支持筛选：
   - 搜索标题、渠道、URL、备注
   - 渠道类型
   - 发布状态
4. 在博客模块的草稿列表下方展示发布记录历史。
5. 在设置模块的发布相关区域复用同一个历史面板。
6. 验证：
   - `CI=true pnpm build`
   - `cargo check`
   - `cargo test`
   - 浏览器检查：博客模块可见发布记录历史、筛选控件，无 console error。

## 非目标

- 不删除发布记录。
- 不编辑发布记录。
- 不做真实外部发布。
- 不做批量导出。
