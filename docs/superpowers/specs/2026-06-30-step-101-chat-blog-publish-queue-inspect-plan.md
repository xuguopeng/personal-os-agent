# Step 101：聊天查看博客发布队列

## 目标

让 `@博客` 聊天指令可以查看已有博客草稿关联的发布 dry-run 队列，例如 `查看《标题》的发布队列`、`《标题》的发布计划现在是什么状态`。

## 小步骤

1. 在 Agent 任务结果里增加发布队列查看结果字段。
2. 增加发布队列查看意图识别，要求包含标题线索或当前草稿，并表达查看发布队列/发布计划状态。
3. 复用标题线索匹配逻辑选择目标草稿。
4. 从执行队列中筛选对应 `chat_blog_publish_plan` dry-run 项。
5. 追加 `publishing_publish_queue_inspect` 流程步骤，记录队列项数量和状态。
6. 在聊天回复里说明队列状态，最多展示 5 条。
7. 跑前端构建、Rust check/test，并用浏览器验证先创建 dry-run，再查看队列。

## 范围

- 只读取本地执行队列。
- 不修改队列状态，不触发外部发布。
- 找不到队列项时明确说明未找到。

## 验收

- 输入 `@博客 查看《某草稿标题》的发布队列` 后，聊天回复展示关联队列状态。
- 底部流程日志出现 `publishing_publish_queue_inspect` 和 `agent.blog.publish.queue.inspect`。
- 构建、Rust 检查和测试通过。
