# Step 46: 发布草稿列表面板

## 目标

把已经保存的发布草稿展示出来。第一版支持查看、搜索、按渠道筛选和复制全文，不做真实发布。

## 小步骤

1. App 增加 `publishingDrafts` 状态。
2. 启动和保存发布草稿后刷新草稿列表。
3. 设置页发布渠道区域下方增加“发布草稿”面板。
4. 草稿面板支持：
   - 搜索标题和正文
   - 按渠道类型筛选
   - 展开查看 Markdown 内容
   - 复制全文
5. 验证：
   - `CI=true pnpm build`
   - `cargo check`
   - `cargo test`
   - 浏览器检查：发布草稿面板可见，无 console error。

## 非目标

- 不真实发布外部平台。
- 不删除草稿。
- 不编辑草稿。
- 不导出密钥、token、cookies 或 secret。
