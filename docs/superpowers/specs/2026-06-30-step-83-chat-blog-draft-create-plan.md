# Step 83: 聊天创建博客草稿

## 目标

让聊天中的 `@博客` 指令真正调用博客模块创建本地发布草稿，不再只是生成模拟流程日志。

## 小步骤

1. `AgentTaskResult` 增加可选 `blogDraft`。
2. `runAgentTask` 在识别到 `module === "blog"` 时创建本地发布草稿。
3. 草稿标题从聊天内容提取，正文优先使用真实模型返回内容，否则使用本地模拟 Markdown。
4. 默认选择已启用发布渠道并写入 frontmatter。
5. 创建草稿后追加 `publishing_draft_create` 流程步骤。
6. 刷新发布草稿列表。
7. 右侧聊天回复提示草稿已生成。
8. 验证：`CI=true pnpm build`、`cargo check`、`cargo test`、浏览器输入 `@博客 测试草稿` 后可见草稿生成和流程日志。

## 非目标

- 不真实发布到网站或公众号。
- 不调用付费图片、视频或外部发布接口。
- 不自动群发公众号文章。
