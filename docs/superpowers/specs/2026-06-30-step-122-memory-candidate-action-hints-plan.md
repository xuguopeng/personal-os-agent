# Step 122：记忆候选列表显示审核命令

## 目标

让 `@记忆 待确认` 的聊天回复不仅列出候选，还直接给出下一步可输入的审核命令。用户看到候选后，可以复制或继续输入 `@记忆 批准候选 N` / `@记忆 拒绝候选 N`，不用猜命令格式。

## 小步骤

1. 更新记忆候选列表回复格式。
2. 每条候选下显示两条命令：
   - `@记忆 批准候选 N`
   - `@记忆 拒绝候选 N`
3. 空列表时不显示命令提示。
4. 构建、Rust 检查、浏览器验证。

## 验证

- `CI=true pnpm build`
- `cargo check`
- `cargo test`
- 浏览器验证：
  - 输入 `@记忆 待确认` 后，回复里出现批准和拒绝命令。
  - 页面控制台没有应用错误。
