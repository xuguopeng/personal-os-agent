# Step 42: 任务日志时间线视图

## 目标

给底部任务日志增加时间线视图。分组视图适合看 Agent、MCP、Skill、模块数据分类；时间线视图适合按真实执行顺序追踪一条任务从聊天到执行结果的完整过程。

## 小步骤

1. `ProcessLog` 增加本地视图模式：
   - 分组
   - 时间线
2. 调用链总览增加视图切换控件。
3. 分组视图保留现有表现。
4. 时间线视图按当前筛选和搜索后的步骤顺序展示。
5. 时间线节点显示：
   - 原始步骤编号
   - 创建时间
   - 步骤卡片
6. 自动展开、搜索、筛选继续在两种视图里生效。
7. 验证：
   - `CI=true pnpm build`
   - `cargo check`
   - `cargo test`
   - 浏览器检查：视图切换可见，流程日志正常，无 console error。

## 非目标

- 不改变数据库。
- 不改变步骤排序。
- 不新增后端命令。
- 不做拖拽、编辑或重排。
