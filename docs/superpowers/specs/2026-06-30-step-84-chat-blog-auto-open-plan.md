# Step 84: 聊天创建博客草稿后自动打开博客模块

## 目标

`@博客` 已经能从聊天创建本地发布草稿。本步让草稿创建成功后自动切换到博客模块，用户可以立刻查看、编辑、检查和导出清单。

## 小步骤

1. 在 `runAgentTask` 创建博客草稿成功后调用 `setActiveModule("blog")`。
2. 保持底部流程日志自动展开。
3. 不改变其他模块的聊天行为。
4. 不真实发布外部平台。
5. 验证：`CI=true pnpm build`、`cargo check`、`cargo test`、浏览器输入 `@博客 ...` 后自动进入博客页且无 console error。

## 非目标

- 不自动打开草稿编辑态。
- 不真实发布到网站或公众号。
- 不调用付费模型或外部发布接口。
