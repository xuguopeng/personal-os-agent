# Step 50: 博客模块接入发布草稿

## 目标

把发布草稿从设置页扩展到左侧“博客”模块。打开博客工作台时，可以直接查看、搜索、复制、下载和管理发布草稿状态。

## 小步骤

1. `Workspace` 为 `moduleKey === "blog"` 增加独立分支。
2. 博客模块顶部显示：
   - 发布草稿数量
   - 待发布数量
   - 发布渠道数量
3. 博客模块复用 `PublishingDraftsPanel`。
4. 博客模块保留 `Toolbar`，后续继续接入新建草稿和发布记录。
5. 验证：
   - `CI=true pnpm build`
   - `cargo check`
   - `cargo test`
   - 浏览器检查：博客模块可见发布草稿面板，无 console error。

## 非目标

- 不真实发布外部平台。
- 不新增草稿编辑器。
- 不删除草稿。
- 不新增后端命令。
