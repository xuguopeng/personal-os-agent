# Step 117：只看待确认开关显示数量

## 目标

在流程日志左侧“只看待确认”开关上显示当前加载任务里的待确认任务数量。用户不必打开开关，也能知道最近任务列表里有多少项需要处理。

## 小计划

1. 复用 `pendingTaskSessionIds.size`。
2. 在“只看待确认”标签旁显示数量：
   - 有待确认任务：`只看待确认 · N`
   - 无待确认任务：`只看待确认`
3. 数量只表示当前已加载的最近任务列表，不代表全库统计。
4. 不改变任务筛选和确认执行逻辑。

## 验证

- `CI=true pnpm build`
- `cargo check`
- `cargo test`
- 浏览器验证：
  - 生成删除待确认任务后，开关显示待确认数量。
  - 打开开关后任务列表仍显示待确认任务。
  - console error 为空。
