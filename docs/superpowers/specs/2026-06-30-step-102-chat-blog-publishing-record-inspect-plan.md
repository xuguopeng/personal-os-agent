# Step 102：聊天查看博客发布记录

## 目标

让 `@博客` 聊天指令可以查看已有博客草稿的本地发布记录，例如 `查看《标题》的发布记录`、`《标题》发到哪里了`。

## 小步骤

1. 在 Agent 任务结果里增加发布记录查看结果字段。
2. 增加发布记录查看意图识别，要求包含标题线索或当前草稿，并表达查看发布记录/发布历史/发到哪里。
3. 复用标题线索匹配逻辑选择目标草稿。
4. 从发布记录表里筛选目标草稿的记录，按时间倒序最多展示 5 条。
5. 追加 `publishing_record_inspect` 流程步骤，记录发布记录数量和摘要。
6. 在聊天回复里说明发布记录状态。
7. 跑前端构建、Rust check/test，并用浏览器验证先记录发布，再查看发布记录。

## 范围

- 只读取本地发布记录。
- 不修改草稿，不修改发布记录，不触发外部平台。
- 找不到记录时明确说明未找到。

## 验收

- 输入 `@博客 查看《某草稿标题》的发布记录` 后，聊天回复展示关联发布记录。
- 底部流程日志出现 `publishing_record_inspect` 和 `agent.blog.record.inspect`。
- 构建、Rust 检查和测试通过。
