# Step 41: 任务日志步骤自动展开策略

## 目标

让任务日志在搜索、排查错误和查看执行结果时更省操作。第一版只做前端展示策略，不改变数据库和步骤写入逻辑。

## 小步骤

1. `ProcessLog` 增加“自动展开”本地开关，默认打开。
2. 自动展开范围：
   - 搜索命中的步骤
   - 错误步骤
   - 执行结果步骤
   - 关键执行节点
3. 用户手动展开的步骤继续保留。
4. 用户手动收起自动展开的步骤时，本轮显示中保持收起。
5. 搜索词、筛选条件、当前任务变化后，清理手动收起记录，避免旧状态影响新视图。
6. 验证：
   - `CI=true pnpm build`
   - `cargo check`
   - `cargo test`
   - 浏览器检查：自动展开开关可见，流程日志正常，无 console error。

## 非目标

- 不保存自动展开偏好到数据库。
- 不新增后端命令。
- 不做滚动定位。
- 不做命中词高亮。
