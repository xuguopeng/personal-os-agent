# Step 107：待确认命令填入聊天框

## 目标

让“待确认操作”里的建议命令可以直接填入右侧 Agent 聊天输入框。第一版只填入，不自动发送，避免误触删除、发布、付费生成或导出。

## 小计划

1. 在主应用状态里保存一个待填入聊天框的命令。
2. 从底部流程日志把建议命令传回主应用。
3. 右侧 AgentPanel 监听待填入命令：
   - 自动打开聊天面板。
   - 把命令写入输入框。
   - 聚焦输入框。
   - 不自动发送。
4. 待确认卡片增加“填入聊天”按钮。
5. 保留“复制确认命令”按钮作为 fallback。

## 验证

- `CI=true pnpm build`
- `cargo check`
- `cargo test`
- 浏览器验证：
  - 普通删除草稿后出现待确认项。
  - 点击“填入聊天”后，右侧聊天输入框出现确认命令。
  - 草稿没有因为填入动作被删除。
  - console error 为空。
