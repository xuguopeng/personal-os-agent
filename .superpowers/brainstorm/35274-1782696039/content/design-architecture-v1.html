正式设计 1/5：整体架构

第一版是一个本地优先的 Personal OS，核心是“可观察的 Agent 调度器”。

Personal OS v1
Modules
小说工作台
音乐播放器
博客记录
漫画入口
表情包入口
无限画布入口
Agent Command Center
聊天区：自然语言 + @模块命令
任务草稿
待确认动作
Visible Process
1. 识别意图
2. 读取相关记忆
3. 选择 skill / MCP
4. 调用模块工具
5. 生成结果
6. 写入日志和记忆候选

核心原则

应用不是“很多页面堆在一起”，而是一个本地桌面个人系统：聊天负责表达意图，模块负责具体工作，右侧流程日志负责把 Agent 的思考和工具调用变成可追踪记录。
