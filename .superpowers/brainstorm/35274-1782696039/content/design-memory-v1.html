Learning Flow
1. 从聊天/操作中发现候选记忆
2. 标记来源和可信度
3. 进入待确认队列
4. 你确认后写入长期记忆
5. 调用时展示“用了哪些记忆”

记忆规则

第一版默认不悄悄写入长期记忆。Agent 可以提出“我是否要记住这个偏好？”，你确认后才保存。每条记忆都有类型、内容、来源、创建时间、更新时间、可信度、启用状态。记忆本地保存，并支持手动导出/导入备份。
